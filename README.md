# Statistical Customer Segmentation System

## Overview

This project develops a statistical learning framework for customer segmentation using high-dimensional consumer behavior data.

By integrating exploratory data analysis, feature engineering, dimensionality reduction, cluster analysis, and customer profiling techniques, the system identifies distinct customer groups and uncovers behavioral patterns that support data-driven decision-making.

The project demonstrates practical applications of statistical learning, data mining, computational statistics, and high-dimensional statistical modeling.

---

## Objectives

* Analyze customer purchasing behavior
* Engineer meaningful customer-level features
* Reduce dimensional complexity using PCA
* Identify customer segments using clustering methods
* Evaluate clustering performance
* Develop actionable customer profiles

---

## Research Question

Can statistical learning and high-dimensional modeling techniques identify meaningful customer segments within complex consumer behavior data?
